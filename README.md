# Website

## 关于项目
本项目是一个基于 Docsocusaurus 的静态网站，用于展示企业技术文档。

## 如何开发

1. 克隆项目
``` bash
git clone https://github.com/stra-air-ltd/docs
```

2. 安装依赖
``` bash
cd docs
npm install
```
3. 启动项目
``` bash
npm run serve
```

## 如何使用

1. 克隆项目

``` bash
git clone https://github.com/stra-air-ltd/docs
``` 

2. 构建项目

``` bash
cd docs
npm run build
```
